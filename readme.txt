Show the Live Cricket Score of all international cricket matches and display in your widget.

== Description ==

Display Live Cricket Score on your website updated every 20 seconds, providing your visitors real time cricket scores that will keep them loving your website and increase your search engine rankings by increasing time on site.
Visit http://www.cricscores.net/news/ for demo and drop me a comment there in case of any problem or issue

== Installation ==
1. Download plugin from http://wordpress.org/plugins/live-cricket-scores-widget
1. Upload folder `live-cricket-scores-widget` to the `/wp-content/plugins/`
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the \'Scorebaord Widget\' Widget and set its width and height from widget menu

All Done
